## Getting Started

Make sure you have [NodeJS](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed.

1. Clone the repo

   ```
   git clone https://github.com/LombaxTech/Language-Exchange-Backend.git
   ```

2. Install your dependencies

   ```
   npm install
   ```

3. Start your app

   ```
   npm start
   ```
